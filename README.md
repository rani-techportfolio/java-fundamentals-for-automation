# Java Fundamentals for Automation Testing

## 📌 Overview
This repository contains core Java programs designed to build a strong foundation for automation testing using Selenium WebDriver.

The focus is on logical programming, problem-solving, and understanding Java concepts required for real-time automation frameworks.

---

## 📚 Topics Covered

### 🔹 Conditional Logic Programs
- Biggest of Three Numbers
- Largest of Two Numbers
- Leap Year Check
- Odd or Even Number
- Positive, Negative or Zero
- Simple Calculator using Switch
- Vote Eligibility Check
- Vowel or Consonant Check

---

### 🔹 Arrays

-  Array Analysis (Sum, Average, Largest & Smallest Element)
-  Check if an Array is Sorted
-  Count Odd and Even Numbers in an Array
-  Reverse an Array
-  Find the Largesr and Second Largest Number in an Array

---

### 🔹 Loops

- Armstrong Number
- Sum of Digits of a Number
- Fibonacci Series
- Sum of First N Natural Numbers
- Inverted Star Pyramid Pattern
- Perfect Number
- Prime Number Check
- Reverse a Number

 ---

### 🔹 String Manipulation

- Count a Specific Character and Word Occurrence
- Count Words
- Check Palindrome String
- String Analysis (Vowel, Consonant, Digit, Space, Special Character Count)

 ---
