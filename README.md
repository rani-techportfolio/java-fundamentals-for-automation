# Java Fundamentals for Automation Testing

## 📌 Overview
This repository contains core Java programs designed to build a strong foundation for automation testing using Selenium WebDriver.

The focus is on logical programming, problem-solving, and understanding Java concepts required for real-time automation frameworks.

---

## 📚 Topics Covered

### 🔹 Conditional Logic Programs
- Biggest of Three Numbers
- Largest of Two Numbers
- Leap Year Check
- Odd or Even Number
- Positive, Negative or Zero
- Simple Calculator using Switch
- Vote Eligibility Check
- Vowel or Consonant Check

---

## 🎯 Purpose of this Repository
- Strengthen Core Java fundamentals
- Build logic required for Selenium automation frameworks
- Practice real interview-based coding questions
- Improve problem-solving skills
